# ⚖️ 劳动者离职补偿计算器（LCC）

> Labor Compensation Calculator — 一款基于《劳动合同法》的离职补偿估算工具

本工具用于帮助劳动者在离职时，快速估算可能获得的经济补偿金、未休年假折算工资、加班费、未签合同二倍工资差额等法定权益。所有计算均在浏览器本地完成，**不会上传或存储任何个人数据**。

🔗 **在线体验**：将本仓库下载后，直接打开 `index.html` 即可使用，无需任何依赖。

---

## ✨ 功能特性

- **📆 工作年限计算**：自动根据入职/离职日期计算工作年限，精确到“满半年按一年算”
- **💰 经济补偿金（N / N+1 / 2N）**：覆盖协商解除、单位单方解除、违法解除、到期不续签、PIP 解除、试用期解除等 10+ 种场景
- **📅 未休年假折算**：支持法定年假与福利年假，自动按 21.75 天折算日工资
- **💸 欠付工资 & 提成款**：可单独填写未结清的工资、奖金、提成
- **⏱️ 加班费计算**：区分工作日延时、休息日、法定节假日三种倍率（150%/200%/300%）
- **📄 未签合同二倍工资**：支持计算未签书面劳动合同的二倍工资差额（最长 11 个月）
- **📈 期权/股权激励参考估值**：提供方向性参考框架，不计入总额（明确提示争议风险）
- **📋 详细名词说明**：内置 N、N+1、2N、社平三倍封顶、代通知金、21.75/20.83 等法律术语解释
- **📱 响应式设计**：完美适配电脑、平板、手机
